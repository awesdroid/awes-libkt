awes-libkt
==========
This is a Kotlin library with some useful extensions for android development

## subpackages
### common
Non-Android stuff

- Dispatchers
- AbstractSerialChain
- MergedData
- MutableLazy

### android
Android stuff

- LiveException
- ActivityHelper
